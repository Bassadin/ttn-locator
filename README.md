# ttnmapper-reader

Part of my [Master's Thesis in summer semester 2023](https://github.com/Bassadin/Master-Thesis-INM)

A simple "microservice" that reads data from TTNMapper and stores it in a PostgreSQL database.

## Installation

```bash
pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
